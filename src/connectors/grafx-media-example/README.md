# GraFx Media example connector

This connector is an example implementation of a GraFx Media connector for the GraFx Environment API. 

It demonstrates how to implement the Connector interface to allow querying, retrieving details, downloading and uploading media

It is derived from the internal GraFx Studio Media connectors but is not intended to be used, nor is there any promise that this connector will keep in sync.

> **_NOTE:_** inside the gitignore there is a staticKey.json, this can be used to store locally a static key for authentication and it is there to make sure it doesn't get accidentally leaked into the repository.
